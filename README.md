# PIV

1. Setup
-------------------------------
The correct virtual environment must be set-up and activated
if the packages are not installed locally.
The dependencies are:
- python
- python-scipy
- python-numpy
- python-progressbar
- python-scikit-image
- cython

All these are availible through the python package installer pip.
